# Superagency Landing Page

Interactive summary of **Superagency: What Could Go Right with AI - and What We Can Do About It** by Reid Hoffman and Greg Beato.

## Features

- Dark tech aesthetic with modal-based navigation
- 3-paragraph book overview
- 11 chapter cards with 5 key bullet points each
- Embedded PDF viewer (PDF.js)

## Live Demo

[View on GitHub Pages](https://charlesmartinedd.github.io/superagency-landing/)

## Local Development

Simply open `index.html` in a browser. No build step required.
